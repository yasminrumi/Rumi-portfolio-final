Chess Game

An interactive chessboard built with HTML, CSS, and JavaScript. Click a piece to select it, then click a square to move it. In this project, I learned how to manipulate the DOM by moving image elements between grid cells and how to manage state variables in JavaScript. Note: this is a simplified version without move validation.

QR Positioning

A QR-code based system that places a marker on a store map after scanning a QR code. The QR code contains JSON data with position (in %) and product info. In this project, I learned how to use third-party libraries (html5-qrcode), parse JSON data, and use CSS absolute positioning with percentage values so the marker stays accurate on any screen size.
